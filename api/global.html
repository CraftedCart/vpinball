Sub QuitPlayer(CloseType: Integer = 0)

Quits the player.

Parameters:

CloseType

Optional: defaults to 0 if not given

0 = Exit player (and also quit Visual Pinball if it started minimized)

1 = Quit Visual Pinball

2 = Quit Visual Pinball immediately without fully cleaning up

Passing any other number will quit Visual Pinball

Function GetBalls() → SAFEARRAY(VARIANT)

Returns an array of all balls.

Example setting all balls on the table to red: 

1Dim Ball
2For Each Ball In GetBalls
3    Ball.Color = &HFF0000
4Next

Returns:

Array of all balls
