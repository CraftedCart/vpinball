Kicker¶

  • Description

  • API reference

Description¶

A kicker object can capture balls and impart forces (kick) them. In Visual Pinball, kickers can also create and destroy balls.

Function DestroyBall() → Integer

Destorys the ball captured by the kicker.

This has no effect if the kicker hasn’t captured any ball

Example of a kicker being used to implement a drain: 

1Sub DrainKicker_Hit()
2    DrainKicker.DestroyBall
3    ' You may also want to play a sound effect, update a "balls in play" counter, spawn a new ball,
4    ' etc.
5End Sub

Returns:

Number of balls destroyed (0 or 1)

Sub Kick(angle: Single, speed: Single, inclination: Single = 0)

Impart a force on the captured ball.

This has no effect if the kicker hasn’t captured any ball

Parameters:

angle

Angle to kick the ball out at, in degrees

0 is towards the back of the table

90 is towards the right

180 is towards the front of the table

270 is towards the left

speed

The speed to kick out at

inclination

Optional: defaults to 0 if not given
